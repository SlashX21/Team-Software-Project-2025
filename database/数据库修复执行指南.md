### 第四步：数据验证 ✅

```bash
# 验证数据导入成功
mysql -uroot -p20020213Lx -Dspringboot_demo -e "
SELECT 'Data import verification:' AS info;
SELECT COUNT(*) as total_products FROM PRODUCT;
SELECT COUNT(*) as total_allergens FROM ALLERGEN;
SELECT COUNT(*) as total_users FROM USER;

-- 验证测试商品是否存在
SELECT 'Test product verification:' AS info;
SELECT bar_code, product_name, brand, category, energy_kcal_100g 
FROM PRODUCT 
WHERE bar_code LIKE '%10001417%' OR bar_code = '10001417' 
LIMIT 3;

-- 显示商品类别分布
SELECT 'Product category distribution:' AS info;
SELECT category, COUNT(*) as count 
FROM PRODUCT 
GROUP BY category 
ORDER BY count DESC 
LIMIT 10;
"
```

**验证标准**：
- ✅ PRODUCT表记录数 > 0（预期约2643条）
- ✅ ALLERGEN表记录数 > 0（预期约105条）
- ✅ USER表记录数 = 8（保留的用户数据）
- ✅ 能找到测试商品（如条码包含10001417的商品）

### 第五步：AI服务配置 🤖

```bash

# 2. 创建Java-MySQL集成配置文件
cat > test_maven_db.env << 'EOF'

ENVIRONMENT=java_integration
DB_TYPE=mysql
JAVA_DB_CONNECTION_STRING=mysql+pymysql://root:20020213Lx@localhost:3306/springboot_demo?charset=utf8mb4
OPENAI_API_KEY=sk-test-mock-key-for-local-development
API_PORT=8001
API_HOST=0.0.0.0
LOG_LEVEL=INFO
JAVA_BACKEND_URL=http://localhost:8080
EOF

# 3. 清理端口并重启AI服务

source test_maven_db.env
uvicorn api.main:app --host 0.0.0.0 --port 8001 --reload
```
